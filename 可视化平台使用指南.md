# 数据可视化平台使用指南

## 平台概述
这是一个基于Web的数据可视化平台，可以将CSV文件快速转换为交互式图表。平台支持多种图表类型，包括折线图、柱状图、饼图和散点图。

## 已创建的CSV文件
平台已预加载以下数据集，可直接使用：

1. **气温数据.csv** - 未来15天最高气温和最低气温数据
2. **阿里巴巴GMV数据.csv** - 2013-2019财年阿里巴巴GMV数据  
3. **网购替代率数据.csv** - 各商品种类网购替代率数据
4. **物流费用数据.csv** - 三家物流公司月度费用数据
5. **汽车速度制动距离数据.csv** - 汽车速度与制动距离关系数据
6. **发电量统计数据.csv** - 2017和2018年全国月度发电量数据
7. **支付宝账单数据.csv** - 支付宝月账单分类数据
8. **霍兰德职业兴趣测试数据.csv** - 职业兴趣测试分数数据
9. **细根生物量数据.csv** - 四个树种不同季节的细根生物量数据

## 使用方法

### 1. 启动可视化平台
双击运行 `start_server.bat` 文件，或执行以下命令：
```bash
python serve_visualization.py
```

### 2. 访问平台
在浏览器中打开：`http://localhost:8000/数据可视化平台.html`

### 3. 选择数据源
- **预定义数据集**：点击左侧"预定义数据集"中的文件名直接加载
- **上传CSV文件**：点击"上传CSV文件"按钮上传自定义数据

### 4. 配置图表
- **图表类型**：选择折线图、柱状图、饼图或散点图
- **X轴数据**：选择作为X轴的列
- **Y轴数据**：选择作为Y轴的列
- **数据系列**：选择要显示的数据列（多选）

### 5. 生成图表
点击"生成图表"按钮，右侧将显示交互式图表

## 图表类型说明

### 折线图
- 适合显示时间序列数据
- 示例：气温变化趋势、GMV增长趋势

### 柱状图
- 适合比较不同类别的数据
- 示例：各商品替代率比较、物流费用对比

### 饼图
- 适合显示各部分占比
- 示例：支付宝账单分类占比

### 散点图
- 适合显示变量间的关系
- 示例：汽车速度与制动距离关系

## 功能特点

✅ **交互式图表** - 支持缩放、拖拽、数据点提示  
✅ **多种图表类型** - 满足不同数据可视化需求  
✅ **数据筛选** - 支持按列筛选显示数据  
✅ **响应式设计** - 适配不同屏幕尺寸  
✅ **数据表格** - 同时显示原始数据表格  
✅ **图表导出** - 支持将图表保存为图片  

## 技术栈
- **前端**：React + Ant Design + ECharts
- **数据格式**：标准CSV格式
- **服务器**：Python HTTP服务器

## 注意事项
- 确保CSV文件使用UTF-8编码
- 数值列应包含纯数字，不含特殊字符
- 文件大小建议不超过10MB
- 支持中文列名和中文数据

## 快速开始示例

1. 点击"气温数据.csv"加载数据集
2. 选择图表类型为"折线图"
3. X轴选择"天数"，Y轴选择"最高气温"
4. 数据系列选择"最高气温"和"最低气温"
5. 点击"生成图表"查看气温变化趋势

现在您可以开始使用这个强大的数据可视化工具了！
